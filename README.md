## Личный сайт — 3D/Universe портфолио

Одностраничный личный сайт **Клима Душкина** c динамичным 3D/Universe‑фоном, интерактивным блоком «Мои навыки» и индексом навыков с сохранением состояния в `localStorage`.

Сайт написан на **чистых HTML/CSS/JS** без фреймворков и может быть развернут на любом статическом хостинге (GitHub Pages, Vercel, Netlify и т.п.).

---

### Стек и особенности

- **HTML**: семантическая вёрстка, разделение на блоки:
  - `hero` — главный экран с именем, описанием и CTA‑кнопкой;
  - `skills` — интерактивный список навыков;
  - `skill-index` — «Индекс навыков» со слайдерами и общим показателем;
  - `contact` — блок с ссылкой для связи в Telegram.
- **CSS (`styles.css`)**:
  - неоновый градиентный фон и вселенная со «звёздами» и «туманностями»;
  - анимации орбов и параллакс‑эффекты;
  - адаптивная верстка под разные устройства.
- **JavaScript (`script.js`)**:
  - плавный скролл к секциям по якорным ссылкам;
  - реакция фона на движение курсора (CSS‑переменные `--mouse-x`, `--mouse-y`);
  - блок «Мои навыки»:
    - добавление/удаление навыков;
    - отметка навыка как изученного;
    - хранение списка навыков в `localStorage`;
  - блок «Мой индекс навыков»:
    - слайдеры по ключевым направлениям (HTML, CSS, JS, React/Vue, LLM/нейросети);
    - вычисление среднего процента и текстовое описание уровня;
    - сохранение значений в `localStorage`.

### Локальный запуск

1. Клонировать репозиторий:

```bash
git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>.git
cd <имя-папки-репозитория>
```

2. Открыть `index.html` в браузере:

- просто дважды кликнуть по файлу **или**
- запустить локальный сервер (пример для Python 3):

```bash
python -m http.server 8000
# затем открыть http://localhost:8000 в браузере
```
